What Is Yuumei's Bot

Yuumei's Bot is a project made by HaruYuumei, this project consist of a Twitch ChatBot that allows interactions between streamer and viewers throught text based games, this project started as a learning project, HaruYuumei wanted to study programming and decided to learn by doing a bot for twitch that allows online and offline chats to interact in a different way then other bots does.

The bot is still something under development so bugs and errors might be seen during your utilization. Keep in mind that this project is also made and maintained by few people, wich means that future updates might take long times to be launched.

Command Alias Description

!setgame {category} !sg {category} Change the current game category of the stream, If the game is not found, it will choose based on the first word
